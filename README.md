# Crime-Against-Women-in-India-Interactive-Power-BI-Dashboard
Built an interactive Power BI dashboard analyzing crime against women in India using NCRB datasets. The dashboard visualizes crime trends, state-wise crime hotspots, and crime category distribution using maps, heatmaps, and advanced DAX measures with interactive navigation and drill-through analysis.
